# 彩色泡泡射击游戏

一个经典的泡泡射击消除游戏，具有流畅的物理效果和精美的视觉设计。

## 游戏特色

- 🎯 精准的射击瞄准系统
- 🎨 7种鲜艳的彩色泡泡
- 💥 连锁消除动画效果
- 📊 实时得分系统
- 🎮 关卡递进设计
- ✨ 平滑的物理碰撞
- 🌟 粒子特效

## 游戏玩法

- 移动鼠标瞄准
- 点击发射泡泡
- 3个或以上相同颜色的泡泡相连会消除
- 消除所有泡泡即可过关
- 泡泡碰到顶部游戏结束
- 每发射8个泡泡，顶部会下降一行

## 技术实现

- 纯 HTML5 Canvas 绘制
- 六边形网格布局系统
- 物理碰撞检测算法
- BFS 图搜索算法检测连通泡泡
- 流畅的抛物线运动轨迹
