# 双人贪吃蛇大作战游戏

经典贪吃蛇的多人对战版本，支持双人同屏对战或挑战AI！

## 游戏特色

- 🎮 双人同屏对战模式
- 🤖 智能AI对手挑战
- 🐍 两条蛇同时竞争
- 🍎 多个食物同时存在
- 💫 吃食物粒子特效
- 📊 实时数据统计
- 🎨 霓虹发光效果
- 💀 完整碰撞检测系统

## 游戏玩法

### 双人对战模式 (PvP)
- **玩家1**: WASD键控制方向
- **玩家2**: 方向键控制
- 争夺场上的食物
- 撞到对方身体即失败

### 单人挑战模式 (PvE)
- **玩家**: WASD键控制
- **AI**: 自动寻找最近食物
- AI会躲避障碍物和玩家
- 挑战AI的智能算法

## 失败条件

- 撞到边界墙壁
- 撞到自己的身体
- 撞到对手的身体
- 最后存活或分数更高者获胜
