# 3D 堆塔游戏 (Tower Block 3D)

这是一个基于 Three.js 开发的 3D 堆塔小游戏。玩家需要通过点击屏幕，让左右移动的方块尽可能完美地堆叠在底座上。

## 游戏玩法

1.  **开始游戏**：点击屏幕任意位置或"开始游戏"按钮。
2.  **堆叠方块**：方块会在底座上方左右移动，当方块与底座对齐时，再次点击屏幕放置方块。
3.  **完美堆叠**：如果方块与底座完全对齐，方块大小保持不变。
4.  **切割**：如果方块没有完全对齐，多出的部分会被切掉并掉落，接下来的方块会变小，难度增加。
5.  **游戏结束**：如果方块完全没有落在底座上，游戏结束。

## 特色

-   **3D 视觉效果**：使用 Three.js 渲染，具有阴影和光照效果。
-   **流畅交互**：针对移动端和桌面端进行了优化，点击/触摸响应迅速。
-   **动态配色**：随着堆叠高度增加，方块颜色会产生渐变效果。
-   **音效**：使用 Web Audio API 生成简单的合成音效，无需额外加载音频文件。

## 技术栈

-   HTML5
-   CSS3
-   JavaScript (ES6+)
-   Three.js (v0.132.2)
