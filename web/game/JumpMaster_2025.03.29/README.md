一个轻松有趣的无尽跑酷游戏，清新的画风和流畅的操作！

## 游戏玩法

- 按空格键或↑方向键跳跃
- 躲避各种障碍物（仙人掌、岩石等）
- 收集空中的金币获得额外分数
- 游戏速度会逐渐增加
- 挑战最远距离记录！

## 障碍物类型

- **仙人掌**: 标准障碍物，带有侧刺
- **岩石**: 三角形岩石，需要精准跳跃
- **双层障碍**: 更高的挑战
