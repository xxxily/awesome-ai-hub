# 色彩消消乐游戏

一个经典的三消游戏，色彩鲜艳，玩法简单但富有策略性！

## 游戏特色

- 🌈 8×8 彩色方块网格
- 💫 流畅的交换和下落动画
- 🔥 连击系统与分数加成
- 🎯 目标分数挑战
- 💡 提示系统（3次）
- 📊 进度条显示
- 🎨 渐变彩虹标题动画
- ⭐ 支持4-5-6连消

## 游戏玩法

- 点击选择一个方块，再点击相邻方块进行交换
- 将3个或更多相同颜色的方块连成一线即可消除
- 每次移动消耗1次机会
- 连续消除可获得连击加成
- 达到目标分数（1000分）即可获胜

## 消除规则

- **3连消**: 30分
- **4连消**: 40分 + 连击加成
- **5连消**: 50分 + 连击加成
- **连击**: 每次连击额外加 5×连击数 分
