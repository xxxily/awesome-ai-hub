## 游戏玩法

- 使用左右方向键或 A/D 键控制飞船移动
- 按空格键发射激光
- 消灭所有敌人进入下一关
- 躲避敌人的子弹
- 敌人生命值随等级提升而增加
- 达到一定等级后敌人数量也会增加
