# 2048数字消除游戏

一个经典的益智数字消除游戏，具有流畅的动画效果和精美的界面设计。

## 游戏特色

- 🎮 流畅的滑动动画
- 🎨 渐变色彩方块
- 📊 得分和最高分记录
- ↩️ 撤销功能
- 🔄 重新开始
- ⌨️ 键盘和触摸控制
- ✨ 平滑过渡动画

## 游戏玩法

- 使用方向键或滑动操作移动方块
- 相同数字的方块会合并成它们的和
- 每次移动后会随机生成新的2或4方块
- 达到2048即可获胜，也可继续挑战更高数字
- 无法移动时游戏结束

## 技术实现

- 纯 HTML5 Canvas 绘制
- 矩阵算法实现方块移动和合并
- CSS3 过渡动画
- 本地存储最高分和游戏状态
- 触摸事件支持移动设备
