# 塔防大战

一个经典的策略塔防游戏，具有多样化的防御塔和敌人类型，以及完整的升级系统。

## 游戏特色

- 🗼 3种不同类型的防御塔
- 👾 多波次敌人系统
- 💰 资源管理系统
- ⬆️ 塔防升级机制
- 🎯 自动瞄准攻击
- 💥 激光和爆炸特效
- 📈 难度递进设计

## 游戏玩法

- 点击空地放置防御塔
- 防御塔会自动攻击范围内的敌人
- 消灭敌人获得金币
- 使用金币建造和升级防御塔
- 阻止敌人到达终点
- 生命值归零游戏结束

## 塔防类型

- 🔫 **机枪塔**: 快速射击，伤害中等
- 🔥 **火炮塔**: 射速慢，高伤害范围攻击
- ⚡ **激光塔**: 持续伤害，穿透攻击

## 技术实现

- 纯 HTML5 Canvas 绘制
- A*路径寻找算法
- 碰撞检测和范围判定
- 状态机管理游戏流程
- 粒子系统实现特效
