一个炫酷的音乐节奏游戏，考验你的反应速度和节奏感！

- 🎹 4条音轨多音符同步
- 💫 完美/良好双重判定系统
- 🔥 连击系统与分数加成
- ⚡ 粒子爆炸特效

- ⏱️ 60秒限时挑战
- 📈 动态难度调整

## 游戏玩法

- 使用 D、F、J、K 键对应四条音轨
- 当音符到达判定区域时按下对应按键
- **Perfect**: ±50ms 内击中，获得 100 分
- **Good**: ±100ms 内击中，获得 50 分
- **Miss**: 超出范围或未击中，连击中断
- 连击越高，分数加成越大（每10连击增加100%）
