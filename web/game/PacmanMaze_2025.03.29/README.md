# 吃豆人迷宫

一个经典的吃豆人迷宫游戏，具有智能追踪的幽灵和完整的游戏机制。

## 游戏特色

- 🎮 经典迷宫设计

- 🍒 奖励水果系统
- 📊 得分和关卡系统
- 🎵 经典游戏体验
- 🌈 炫彩视觉效果

## 游戏玩法

- 使用方向键控制吃豆人移动
- 吃掉迷宫中所有的豆子过关
- 吃能量豆可以反击幽灵
- 躲避幽灵追踪
- 收集水果获得额外分数
- 被幽灵抓到会失去生命

## 游戏元素

- 🟡 **吃豆人**: 玩家角色
- 👻 **幽灵**: AI敌人，会追踪玩家
- 🔵 **能量豆**: 吃下后可以反击幽灵
- 🍒 **水果**: 额外奖励分数

## 技术实现

- 纯 HTML5 Canvas 绘制
- A*路径寻找算法
- 状态机控制幽灵行为
- 碰撞检测系统
- 平滑动画过渡
